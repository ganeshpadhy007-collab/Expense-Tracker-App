# 💰 Expense Tracker App

A simple and intuitive web app to track your daily income & expenses. Built to help users manage personal finances and visualize spending habits.

### 🚀 Live Demo
🔗 https://ganeshpadhy007-collab.github.io/Expense-Tracker-App/


### ✨ Features
- **Add/Delete Transactions**: Income aur expense dono add karo
- **Category Wise Filter**: Food, Travel, Bills, Shopping etc.
- **Monthly Summary**: Total income, expense & balance auto calculate
- **Data Visualization**: Pie chart & bar graph se spending pattern dekho
- **Responsive Design**: Mobile & desktop dono pe smooth chale
- **Local Storage**: Data browser me save hota hai, refresh pe delete nahi hoga

### 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript / React.js
- **Charts**: Chart.js
- **Styling**: Tailwind CSS / Bootstrap
- **Deployment**: Vercel / Netlify
